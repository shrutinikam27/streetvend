# 🚀 StreetVend – Empowering Street Vendors with Smart Supply Solutions

## 🌍 Overview

**StreetVend** is a MERN stack-based web application designed to help street food vendors connect with trusted suppliers, compare prices, and manage orders efficiently.

The platform aims to solve real-world problems faced by local vendors by providing a **digital supply chain solution** that is simple, affordable, and accessible.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/streetvend.git
cd streetvend
```

### 2️⃣ Install dependencies

#### For Backend

```bash
cd backend
npm install
```

#### For Frontend

```bash
cd frontend
npm install
```

### 3️⃣ Setup Environment Variables

Create a `.env` file in backend:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

### 4️⃣ Run the application

#### Backend

```bash
npm start
```

#### Frontend

```bash
npm start
```
